# HI-re M3 Vercel Site v3

Static Vercel-ready product microsite.

## Deploy
- Upload this folder or connect the repo to Vercel.
- Framework Preset: **Other**
- Build Command: leave blank
- Output Directory: leave blank

## Notes
- Front configurator uses `front-base.webp` as a persistent base layer.
- Finish overlays sit on top of the base and fade in/out.
- Rear view disables finish selections.
- Product shots are separate large-format sections.
